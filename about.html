Пример отправки запросов

Расписание преподавателя (упрощенное): Кафедра-группа-номер_учебной_недели-формат упрощенный(1)
parse_miuby_schedule.php?kafedra=Информационных%20технологий&teacher=Жуковский%20В.С.&week=2&weekf=1

Расписание преподавателя (полное): Кафедра-группа-номер_учебной_недели
parse_miuby_schedule.php?kafedra=Информационных%20технологий&teacher=Жуковский%20В.С.&week=2

Учебные недели преподателя: Кафедра-ФамилияИО
parse_miuby_schedule.php?kafedra=Информационных%20технологий&teacher=Жуковский%20В.С.

Преподаватели на кафедеде: Кафедра
parse_miuby_schedule.php?kafedra=Информационных%20технологий

Группы на специальности:
parse_miuby_schedule.php?specialty=Информационные%20системы%20и%20технологии%20(в%20управлении)

Учебные недели студента : Кафедра-группа
parse_miuby_schedule.php?specialty=Информационные%20системы%20и%20технологии%20(в%20управлении)&group=91201з

Учебные недели студента : Группа
parse_miuby_schedule.php?group=91201з

Учебные недели студента (json): Группа
parse_miuby_schedule.php?group=120605тз&json=1

Расписание студента (полное): Кафедра-группа-номер_учебной_недели
parse_miuby_schedule.php?specialty=Информационные%20системы%20и%20технологии%20(в%20управлении)&group=91201з&week=15

Расписание студента (упрощенное): Кафедра-группа-номер_учебной_недели-формат упрощенный(1)
parse_miuby_schedule.php?specialty=Информационные%20системы%20и%20технологии%20(в%20управлении)&group=91201з&week=15&weekf=1

Расписание студента (упрощенное): Кафедра-группа-номер_учебной_недели-формат упрощенный(1)(json)
parse_miuby_schedule.php?specialty=Информационные%20системы%20и%20технологии%20(в%20управлении)&group=91201з&week=27&weekf=1&json=1

Новости университета (в json-формате):
parse_miuby_schedule.php?get_news=1

Максимальный номер учебной недели, на которое составлено расписание:
parse_miuby_schedule.php?weekn=1

Номер текущей учебной недели и максимальный номер учебной недели, на которое составлено расписание (в json-формате: [currentWeek (int),maxWeekSchedule(int)]):
parse_miuby_schedule.php?weekn=3

Номер текущей учебной недели:
parse_miuby_schedule.php?weekn=2

Список кафедр:
parse_miuby_schedule.php?list_kafedra=1

Список специальностей университета:
parse_miuby_schedule.php?specialities=1

Список всех преподавателей университета:
parse_miuby_schedule.php?allteachers=1

Список всех преподавателей университета (в кавычках, через запятую):
parse_miuby_schedule.php?allteachers=2

Список всех учебных групп университета:
parse_miuby_schedule.php?groups=1

Список всех учебных групп по специальностям университета:
parse_miuby_schedule.php?groups=1&groupsspec=1

Список кафедр (json raspisaniye-vuzov.ru):
parse_miuby_schedule.php?get_faculties=1

Список всех учебных групп университета(json raspisaniye-vuzov.ru):
parse_miuby_schedule.php?get_groups=1

Список специальностей университета (json raspisaniye-vuzov.ru):
parse_miuby_schedule.php?get_spec=1
/raspisaniye-vuzov.ru/get_faculties

Список учебных групп по специальностям университета (json raspisaniye-vuzov.ru):
parse_miuby_schedule.php?get_groups=1&specialty=Информационные%20системы%20и%20технологии%20(в%20управлении)
parse_miuby_schedule.php?get_spec_group=1
/raspisaniye-vuzov.ru/get_groups?faculty_id=Современные%20иностранные%20языки%20(перевод)

Расписание студента (json raspisaniye-vuzov.ru): Группа
parse_miuby_schedule.php?get_schedule=1&group_id=120605тз
/raspisaniye-vuzov.ru/get_schedule?group_id=121201с
